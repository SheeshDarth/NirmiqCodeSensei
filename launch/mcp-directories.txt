Name:
NirmiqLearn OS

Category:
Education / Developer Tools

Short description (one line):
Local-first learning OS for AI-assisted developers — 10 MCP tools for Claude Code, Cursor, and Windsurf.

Full description:
NirmiqLearn OS is a local-first learning OS that connects to Claude Code, Cursor, and Windsurf via MCP. Log debug sessions, generate explain-back questions, map DSA/CS concepts, and track daily learning — entirely on your machine. Zero telemetry, zero cloud, SQLite only. 7 free tools (no key needed) + 3 AI-powered Pro tools (BYOK — requires your own Anthropic key and a Pro license).

GitHub:
https://github.com/SheeshDarth/NirmiqLearnOS

Transport:
stdio

Install command:
npm run mcp   (after: git clone + npm install + npm run db:migrate)

Full install steps:
git clone https://github.com/SheeshDarth/NirmiqLearnOS.git
cd NirmiqLearnOS && npm install
npm run db:generate && npm run db:migrate
npm run mcp

Tags:
learning, education, local-first, sqlite, dsa, debugging, explain-back, mcp, developer-tools, student

2. Anthropic MCP Directory
Submit at: https://console.anthropic.com/mcp  (or via the manifest file)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Use the mcp-manifest.json file in the repo root.
It contains all fields needed for submission.

Or fill manually:

Server name: nirmiqlearn
Display name: NirmiqLearn OS
Description: Local-first learning OS. Log debug sessions, generate explain-back questions, map DSA concepts — inside Claude Code. Zero cloud.
Homepage: https://github.com/SheeshDarth/NirmiqLearnOS
Transport: stdio

MCP config (for Claude Code):
{
  "mcpServers": {
    "nirmiqlearn": {
      "command": "npm",
      "args": ["run", "mcp"],
      "cwd": "/absolute/path/to/NirmiqLearnOS"
    }
  }
}
