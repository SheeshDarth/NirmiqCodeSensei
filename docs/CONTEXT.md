# Context — NirmiqCodeSensei

## Why This Product Exists

NirmiqCodeSensei is created for students who use AI to build projects but still want to become genuinely skilled.

AI coding tools can generate code quickly, but students may lose:
- Analytical thinking
- Debugging ability
- Code comprehension
- DSA foundation
- Confidence in interviews
- Ability to explain their own work

This product is not meant to stop AI usage.
It is meant to make AI-assisted building educational, disciplined, and skill-building.

## Core Belief

A student should be allowed to use AI, but must be forced to understand what AI helped create.

## Brand Meaning

NIRMIQ should feel like:
- Personal intelligence system
- Learning operating system
- Project understanding engine
- Academic mentor
- Debugging discipline tool
- Anti-blind-vibecoding system

## Product Personality

NirmiqCodeSensei should be:
- Serious
- Clean
- Premium
- Academic
- Engineering-focused
- Local-first
- Non-generic
- Minimal but powerful
- Strict when needed

It should not feel like:
- A childish learning app
- A generic AI SaaS landing page
- A ChatGPT clone
- A Notion clone
- A basic todo app

## User Emotion

The user should feel:
- “I finally understand my project.”
- “I can explain this in viva.”
- “I am not just copying AI.”
- “I am becoming dangerous in a good way.”
- “I am building proof of skill.”

## Primary Modes

### 1. Learn My Project
Break down a project into modules, concepts, files, and explanation checkpoints.

### 2. Explain-Back
Ask the user questions until they can clearly explain the project.

### 3. Debug Lab
Turn every bug into a learning asset.

### 4. DSA Bridge
Connect real project features to DSA/fundamental concepts.

### 5. Exam/Viva Prep
Convert project understanding into viva answers, interview answers, and revision notes.

## Development Philosophy

Build small.
Commit often.
Use docs as memory.
Use Graphify for context retrieval.
Use LLM Council only for important decisions.
Never dump the whole project into Claude unless necessary.
